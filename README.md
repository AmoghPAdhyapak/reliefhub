# ReliefHub

## The Right Aid. The Right Place. The Right Time.

ReliefHub is an AI-powered disaster coordination platform designed to help citizens, volunteers, NGOs, and emergency response teams coordinate relief efforts efficiently during disasters.

## Problem

During floods, cyclones, landslides, and other emergencies, aid often reaches affected locations late due to:

* Lack of centralized coordination
* Duplicate aid distribution
* Poor resource allocation
* Limited visibility of affected regions
* Communication gaps between citizens and NGOs

As a result, some areas receive excessive aid while others receive none.

## Solution

ReliefHub provides a centralized disaster-response platform that:

* Collects emergency SOS reports
* Uses AI-based priority analysis
* Assigns NGOs based on need and availability
* Generates response routes
* Tracks relief delivery
* Visualizes affected areas on an interactive map

The goal is to ensure aid reaches the right place at the right time.

## Key Features

### Help & Emergency Portal

* Emergency SOS reporting
* Food assistance requests

* Flood reporting
* Location sharing

### Coordination Center

* Live disaster map
* Priority Engine
* NGO assignment system
* Resource allocation
* Duplicate aid detection
* Route guidance

### Weather Radar

* Rainfall monitoring
* Flood risk analysis
* Storm alerts
* Weather visualization

### AI Priority Engine

Priority scores are calculated using:

* Geographic analysis
* Volunteer reports
* NGO reports
* Crowd verification
* Disaster severity
* Previous aid history

This helps identify Critical, Moderate, and Covered regions.

### Real & Simulation Modes

#### Real Mode

Displays verified incidents and real emergency data.

#### Simulation Mode

Allows demonstration of disaster scenarios such as:

* Floods
* Cyclones
* Heavy Rain Events
