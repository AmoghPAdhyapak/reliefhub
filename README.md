# ReliefHub

## The Right Aid. The Right Place. The Right Time.

ReliefHub is an AI-powered disaster coordination platform built to help citizens, volunteers, NGOs, and emergency response teams coordinate relief operations efficiently during natural disasters and emergencies.

The platform combines live reporting, AI-powered prioritization, NGO coordination, weather monitoring, route guidance, and disaster simulations into a single unified ecosystem.

---

# Problem Statement

During disasters such as floods, cyclones, landslides, and severe weather events, relief efforts often suffer from:

- Delayed response times
- Poor communication between stakeholders
- Duplicate aid distribution
- Lack of visibility into affected areas
- Inefficient resource allocation
- Difficulty identifying high-priority locations

As a result, some communities receive excessive aid while others receive little or none.

---

# Our Solution

ReliefHub provides a centralized disaster management platform that:

- Collects emergency SOS requests
- Validates incidents
- Prioritizes affected regions
- Assigns NGOs intelligently
- Tracks aid delivery
- Generates navigation routes
- Monitors weather conditions
- Simulates disaster scenarios for testing

Our goal is simple:

**Ensure that aid reaches the right place at the right time.**

# Key Features

## 🌍 Multi-Language Accessibility

ReliefHub supports multiple Indian languages to ensure accessibility across different regions.

Features:

- Language selection from homepage
- Native language display
- Localized user experience
- Easy accessibility for non-English users

---

## 🚨 Help & Emergency Portal

The Help Portal allows citizens and affected individuals to report emergencies quickly.

Features:

- Emergency SOS Reporting
- Food Assistance Requests
- Medical Assistance Requests
- Flood Reporting
- GPS-Based Location Detection
- Manual Location Entry
- Emergency Contact Support

Users can either:

- Use Current Location
- Enter Location Manually

This ensures accessibility even when GPS is unavailable.

---

## 🗺️ Live Disaster Dashboard

The Live Dashboard acts as the operational command center.

Displays:

- Villages Affected
- Families Requiring Assistance
- Active NGOs
- Relief Resources Delivered
- Regional Monitoring Status
- Disaster Severity Information

The dashboard provides a quick overview of ongoing relief activities.

---

## 🧠 AI Priority Engine

One of the core components of ReliefHub.

The Priority Engine calculates urgency using:

- Geographic Analysis
- Volunteer Reports
- NGO Reports
- Crowd Verification
- Disaster Severity
- Historical Aid Records

Priority Categories:

🔴 Critical

🟠 Moderate

🟢 Covered

This helps ensure that aid is allocated efficiently.
